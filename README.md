# wisemuffin-charts

## Install

```bash
npm install --save wisemuffin-charts
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'wisemuffin-charts'
import 'wisemuffin-charts/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
